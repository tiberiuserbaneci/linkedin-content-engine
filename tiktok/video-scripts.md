## Production Notes
- Script 1: talking-head delivery works. Cut every 4-5 words in editing for fast rhythm.
- Script 2: requires screen capture of Mission Control or phone showing the dashboard. Film the screen, then narrate over it.
- Script 3: no b-roll needed. Straight to camera. Add text overlays for model names (Claude / GPT-4o / Gemini) to reinforce the routing concept visually.
- For all three: no intro, no outro. Hook → content → CTA. First frame must be the hook sentence.
